## Description
[On the Understandability of MLOps System Architectures](https://ieeexplore.ieee.org/stamp/stamp.jsp?tp=&arnumber=10440483)

This article explores the wide variety of architectural descriptions of MLOps systems. It investigates the difference between informal textual descriptions with informal graphical MLOps system architecture representations and semi-formal MLOps system diagrams with a controlled experiment on 63 participants. Data was collected based on correctness and time for different MLOps system descriptions. We will present the information from the paper. 

**Relevance**
This article is relevant to MLOps because it is based on creating MLOps systems. It is important to know the effectiveness of architectural descriptions so that these MLOps practices can continue to be developed and understood. This is also important to DevOps because MLOps is built on DevOps principles with Machine Learning activities.

## Description
[Keep](https://github.com/keephq/keep) is an open source AIOps alert management platform. It can filter and improve readability of alerts and thereby reduce noise and make incident management simpler and more effective
Keep can use various LLM backends, including OpenAI, DeepSeek and Grok.
In our demo we will demonstrate how to create a monitoring workflow for a sample project using Keep. 
We plan to show how a worklow is defined by a yaml configuration file, how to select and switch LLM backend 
and how the LLM filters alerts and defines their severity. We will also show how to use Common Expression Language to further improve alert filtering based on LLM generated information.

Our goal is to highlight how AIOps in general and Keep in particular can be used for noise reduction and contextual enrichment and thereby make monitoring leaner and more effective.

**Relevance**

Alerts and notifications are vital for observability in DevOps work. However, they must be filtered to avoid alert fatigue and ensure that the DevOps team focuses on the most important information. Keep can be configured to do just that - provide the relevant alerts for DevOps.
